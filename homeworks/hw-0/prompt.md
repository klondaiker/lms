# Список промтов

## Первый промпт
Сгенерируй rails приложение 8 версии на ruby 4 версии \
Стек: postgresql, hotwire, tailwind \
Ограничения:
1) никаких лишних гемов
2) используй максимально дефолтный инструментарии для rails (autotest, camal и тд)
Критерии готовности:
1) Есть дефолтная структура rails-приложения
2) bundle install прошел без ошибок
3) rails s запускается без ошибок
4) Дефолтная страница успешно открывается на localhost

## Второй промпт
Сделай регистрацию и авторизацию пользователя \
Стек: используй гем sorcery \
Шаги:
1) Создай модель User с логином и паролем
2) Создай страницы регистрации и авторизации пользователя
3) Регистрация максимально простая - ввод логина и пароля, без проверок и отправок писем на email 
4) Если логин при регистрации уже существует, то выводи ошибку
5) После успешной регистрации сделай редирект на авторизацию
6) После успешной авторизации должно вести на чистую страницу с Привет, login - корень root приложения
8) Если пользователь не авторизирован - то делай редирект на авторизацию
7) Создай автотесты на контроллеры

Критерии готовности:
1) тесты проходят успешно
